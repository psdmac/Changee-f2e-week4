Changee Learning Path - 讓網頁動起來

在網路的時代，我們在開始學習新領域時，可以透過 Google 找到許多學習的「資訊」，然而過多的資訊反而無法判斷什麼才是適合自己的「資源」。現有的學習資源非常豐富，但卻散落在各處，使得「自我學習」時總是花費大量時間在搜尋適合自己的資訊。因此我們希望能夠建立 Changee Learning Path「學習路徑平台」，整合資源與資訊，透過「學習分類帽」、「學習軌跡」、「實體資源」三個主要項目，建立方便自學的「學習路徑平台」。

jQuery

開始前要知道

  • HTML - 內容、結構
  • CSS - 樣式
  • JavaScript - 行為

jQuery Package 裝了沒？

為什麼用 jQuery

  • 簡單好寫
  • 跨瀏覽器（各瀏覽器實作 JavaScript 方法不同）
  • 對於舊瀏覽器有好的支援度

jQuery 口號：write less, do more.

jQuery 可以做

  • 尋找 - 尋找 HTML 文件裡的元素
  • 改變 - 改變 HTML 內容
  • 監聽 - 反應使用者的動作
  • 動畫 - 做出移動、漸變效果
  • 溝通 - 透過網路，動態載入內容

DOM

Document Object Model

  • 由瀏覽器產生的樹狀結構
  • 我們可以利用 JavaScript 快速找到 HTML 元素

jQuery 如何搜尋 DOM

  • 使用 jQuery 前綴
    jQuery(document)
  • 使用 $ 前綴
    $(document)
  • 就像使用 CSS 選取器一樣
  • 選取 HTML 標籤
    $('h1')
  • 選取 Class 
    $('.brand')
  • 選取 ID 
    $('#logo')

jQuery Object

  • 用 jQuery 選取的 DOM 叫 jQuery Method
  • 有非常多的 Method 可以用
  • .html() 就是一種 Method

							$('.brand').html('Hello, World!');

開始使用 jQuery

下載 jQuery

  • v1.11.1 - 第一版支援舊瀏覽器
  • v2.1.1 - 新版本，檔案較小，不支援舊瀏覽器

引入 jQuery

  • 用 script 標籤引入 HTML 中 
    <script src="jquery.min.js"></script>
  • 引入 CDN 檔案，免下載 
    <script src="//code.jquery.com/jquery-1.11.0.min.js"></script>

撰寫 jQuery

  • 引入外部 js 檔案 
    <script src="application.js"></script>
  • 在 HTML 的 script 標籤內撰寫（建議放在 </body> 之前）
    <script type="text/javascript">
  alert("Hello World!");
</script>

切換選單效果

JS Bin

$(this)

  • 只會選取你點的那個
  • $(this) 是 jQuery Object
  • 與 JavaScript 裡面的 this 不同
  • this 的觀念很重要，但初學者可以不要管

回到頁首

JS Bin

.scrollTop(value)

  • 取得目前距離頂端有多遠
  • 數值帶入，就可以直接到指定位置
  • console.log() 可以在開發工具中印出你想要看到的東西

.animate(properties, options)

  • properties 帶入 CSS 的樣式效果
  • options 帶入變化時間
  • 耗效能，已經要漸漸被 CSS3 取代
  • 舊款 IE 瀏覽器就靠它
  • .stop() 可以暫停動畫

.fadeIn(), .fadeOut()

  • 淡出、淡入效果，掌管透明度
  • 有快慢、秒數可以設定
  • 跟動畫差不多，.stop() 可以中斷效果
  • 相似的還有 .fadeTo(), .fadeToggle()

左側選單

JS Bin

變數 var

  • JavaScript 是弱型別語言，不需要宣告變數型別
  • 使用變數要宣告，不然變成全域變數
  • 把會重複利用到的 jQuery Object 設成變數，節省效能

.hasClass(<className>)

  • 判斷 DOM 有沒有那個 Class
  • 回傳 true 或 false
  • 用來判別畫面狀態，搭配 if-else 使用

event.preventDefault()

  • 防止預設的行為觸發
  • 常用在把超連結當按鈕，防止 href 屬性
  • 還有許多 event
  • event.stopPropagation(), event.pageX, event.target
